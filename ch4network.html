Networking and the Internet
Term: Definition:
LAN Local area network
MAN Metropolitan Area Network
WAN Wide area network
Bus Network Machines in a network communicating directy with eachother over a common bus
Star Network Machines in a network communicating indirectly with eachother through an intermediary central machine
CSMA/CD Ethernet Carrier Sense, Multiple Access with Collision Detection, dictates that each message be broadcast to all the machines on the bus
CSMA/CA Wifi Carrier Sense, Multiple Access with Collision Avoidance, designed to avoid collisions and may not eliminate them completely
Repeater A device that passes signals back and forth between the two original buses without considering the meaning of the signals
Bridge Connects two buses, looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side
Switch Essentially a bridge with multiple connections, allowing it to connect several buses rather than two, produces a netwrok consisting of several buses extending from a swtich, as spokes on a wheel
Router Special purpose computers used for forwarding messages, handles the connection betweeen networks to form an internet
Gateway The "point" at which one network is linked to an internet, it serves as a passageway between the network and the outside world
P2P: peer-to-peer Processes that provide service to and recieve service from each other (instant messaging)
Distributed System Consist of software units that execute as processes on different computers
Cloud Computing Whereby huge pools of shared computers on the network can be allocated for use by clients as needed
Tier 1 Consist of very high-speed, high-capacity, international WANs, backbone of the internet
Tier 2 Connects to the tier-1 ISPs that tend to be more regional in scope and less potent in their capabilities
Tier 3 Also called an "access", essentially an independent Internet, operated by a single authority that is in the business of supplying Internet access to individual homes and businesses
ICANN ICANN Internet Corporation for Assigned Names and Numbers, nonprofit corporation established to coordinate the Internets's operation
DNS Domain Name System, Internet-wide directory system
TLD: Top-Level Domain Suffic following the period that is used to reflect the domain's classification (.edu,.com,.gov)
Country-Code TLDs World Standards Two-letter TLDs for each specific country (au.,.ca)
FTP File Transfer Protocol, an application for listing and copying files across the network
HTTP Hyper Text Transfer Protocol, powerful webpage that handles traditional network applications
SMTP Simple Mail Transfer Protocol, defines a way that two computers on the network may interact when transmitting an email message from one host to the other
URL Uniform Resources Locator, a unique adress used to locate and retrieve documents on the web
Client-Side Activities Acitivities performed by a client (such as a browser)
Server-Side Activities Activities performed by a server (such as a webserver)
Application Layer A message originates in the application layer, consists of those software units such as clients and servers that use Internet communication to carry out their tasks
Transport Layer Accepts messages from the application layer and ensures that the messages are properly formatted for transmission over the Internet
Network Layer Decides in which direction a packet should be sent at each step along the packet's path through the Internet
Link Layer In charge of recieving and transmitting the packets
Packet-Switching Network Digital data is sent by breaking the data into blocks of bits called packets, which contain both the data being transmitted and control information for routing the data
Fault Tolerant A computer system that continues working even when there is something wrong with the hardware or software
Redundant The use of multiple copies of information or resources to ensure reliability
Malware Many attacks incorporate the use of malicious software, collectively called malware
Virus Software that infects a computer by inserting itself into programs that already reside in the machine
Worm Autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers
Trojan Horse A program that enters a computer system disguised as a desireable program, such as a game or useful utility package, that is willingly imported by the victim
Spyware Software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack
Phishing A technique of obtaining information explicitly by simply asking for it
DDoS Attack Distributed Denial of Service, The work of generating the attacks is distributed over many hosts
Firewall Program that uses a primary prevention technique that filters traffic passing through a point in the netwrok
Spam Filter Firewalls designed to block unwanted emails
Proxy Server Software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server
Antivirus Software Used to detect and remove the presence of known viruses and other infections
Cryptography The study of sending and recieving messages securely in the presence of adversaries
Public Key Encryption Method of encrypting data with two different keys and making of the keys public, available to anyone, and the other is a private key
