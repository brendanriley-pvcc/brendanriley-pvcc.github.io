All About GitHub

Q: What is a version control system? A: Software that helps the developer team to efficiently communicate and manage/track all the changes that have been made to the source code along with the information like who made and what changes have been made
Q: What is Git? A: Software for tracking changes in any set of files, usually used for coordinating work among programmers collaboratively developing source code during software development
Q: What is GitHub? A: GitHub is a developer platform that allows developers to create, store, manage, and share their code
Q: What is the difference between Git and GitHub? A: Git is the version control system itself, while GitHub is a platform that hosts Git repositories and provides additional collaboration and project management features
Q: Who started GitHub and how was it started? A: GitHub was founded by Tom Preston-Werner, Chris Wanstrath, and PJ Hyett as a web-based platform for hosting repositories and facilitating collaboration among developers
Q: What company owns it now? A: Microsoft purchased GitHub in 2018
Q: How much does a GitHub account cost? A: GitHub free for individual developers/ GitHub Team starts at $3.67 per user, per month
Q: What is the Octocat? A: GitHub's logo of a feline-cephalopod hybrid

Git/GitHub terms:

  • Repository: A central location in which data is stored and managed
  • Commit: An individual change to a file or set of files
  • Fork: A new repository that shares code and visibility setting with the original respository
  • Push: Updates the remote branch with local commits, similar to update or publish
  • Pull Requests: A request you make to the project maintainers to have them review and merge your changes into the main codebase
  • Workflows: A configurable, automated process that executes one or more actions
  • Issues: A bug tracking system built into the GitHub service that allows you to track your work, feedback, tasks, or bugs for a project
  • Raw Button: Allows you to view a file in its raw form, without any HTML formatting
  • Blame Button: GitHub feature designed to help you determine who made changes to a file
